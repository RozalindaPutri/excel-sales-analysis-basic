# 📊 excel-sales-analysis-basic
## 🚀 Project Overview
This project demonstrates a **basic data analysis workflow using Microsoft Excel** to transform raw sales data into meaningful business insights.
The analysis covers essential steps such as **data cleaning, formatting, calculation, and simple analysis** to understand sales performance and profitability.

## 🔑 Key Features
* 🧹 Data Cleaning & Formatting
* 🧮 Basic Excel Formulas (SUM, AVERAGE, COUNT, etc.)
* 🔗 Text Functions (LEFT, RIGHT, MID, CONCAT)
* 🎯 Conditional Logic (IF, AND, OR)
* 💰 Profit Margin & Revenue Calculation
* 🆔 Automated Order ID Generation
* 📊 Basic Data Analysis using Sort & Pivot Table

## 📊 Analysis Insights
* 📈 Sales data can be transformed into **clear performance metrics**
* 💰 Profit margin calculation helps identify **high-profit transactions**
* 🛍️ Analysis reveals **top-performing products and sales trends**
* ⏱️ Data can be used to evaluate **operational efficiency (e.g., delivery time)**

## 🔍 What This Analysis Reveals
🔹 Sales performance overview
🔹 Profitability of each transaction
🔹 Best-selling products
🔹 Average profit insights
🔹 Structured and clean dataset ready for further analysis

## 🛠️ Tools Used
* Microsoft Excel
* Excel Functions (SUM, AVERAGE, IF, COUNT, etc.)
* Data Cleaning & Formatting
* Pivot Table
